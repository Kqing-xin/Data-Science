## 简介

该文件夹包含了多个爬虫单文件，使用了多种方式获取信息，包括爬取评论信息、壁纸图片、异步爬取等内容。

## 使用方法

- 确保Python环境已经安装，并安装所需的第三方库（如requests、BeautifulSoup、selenium等）。
- 打开终端，进入该文件夹所在路径。
- 执行相应的爬虫代码文件，如`python comment_spider.py`。
- 配置相关信息，如保存方式、路径等。
- 等待程序执行完毕，查看相应的结果文件。

## 注意事项

- 请勿滥用爬虫程序，以免违反相关法律法规。
- 如需对爬虫程序进行修改，建议备份原有代码，并在修改前仔细阅读代码注释，以免对程序造成不必要的影响。
